# AI/ML Final Deep Study Studio

A standalone interactive study site for the final-exam scope from unsupervised learning through generative AI.

## Open the site

- Open `index.html` directly in a browser, or serve the folder locally for a more normal web workflow.

## What is included

- `index.html`: app shell
- `styles.css`: layout and visual design
- `app.js`: reader, practice, flashcards, search, and concept-lab behavior
- `content-data.js`: bundled study content used by the site
- `figures/`: chapter diagrams and per-subchapter visual banners

## Study modes

- `Dashboard`: roadmap and topic atlas
- `Reader`: long-form section-by-section lessons
- `Practice`: workbook questions with answer reveal and confidence tracking
- `Cram`: compact review cards
- `Flashcards`: glossary, comparison, and workbook recall cards
- `Concept Lab`: interactive demos and calculators
- `Glossary`: term lookup
- `Search`: cross-topic search

## Notes

- The site is committed as a ready-to-use static bundle.
- Math rendering uses MathJax from a CDN, so equation typesetting needs internet access unless an offline bundle is added later.
